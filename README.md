# 떡상농장 웹 프로토타입 v0.2

SNS에서 아직 뜨지 않은 영상 링크를 발견하고, 제한된 4개 슬롯에 심은 뒤, 가상 조회수 변화에 따라 수확하거나 갈아타는 핵심 루프를 테스트합니다.

## 바로 실행

`index.html`을 브라우저로 열면 별도 설치 없이 동작합니다.

로컬 서버로 실행하려면 이 폴더에서 다음 명령을 사용합니다.

```powershell
npm start
```

그다음 `http://127.0.0.1:4173`을 엽니다.

전체 로직 검증은 다음 명령으로 실행합니다.

```powershell
npm run check
```

## 현재 구현된 것

- URL을 붙여넣거나 샘플 링크를 선택해 포지션 생성
- 링크마다 결정되는 6종류의 가상 성장 곡선
- 동시에 보유 가능한 4개 슬롯
- 밭이 찼을 때 기존 포지션 수확 후 갈아타기
- 씨앗값을 100% 반환하는 무손실 수확
- 같은 유저가 수확한 링크에 다시 진입하지 못하는 일회성 발견
- 심을 때 확정되는 발견 순번과 이후 유입된 발견자 수
- `+1H`, `+6H` 테스트 시계와 24시간 자동 수확
- 진입 조회수, 성장률, 조기 발견 배수, 예상 수확가
- 수확 등급과 공유 문구
- 브라우저 로컬 저장 및 초기화

## 의도적으로 제외한 것

- 실제 YouTube·Instagram·TikTok API
- 로그인과 서버 데이터베이스
- 확정된 농작물 아트
- 유저 간 거래와 현금성 보상

현재 버전은 API 가능성이 아니라 `발견 → 슬롯 선택 → 기다림 → 갈아타기 → 수확`이 재미있는지 확인하기 위한 프로토타입입니다.

버전별 변경 내용은 [`CHANGELOG.md`](./CHANGELOG.md)와 앱 하단의 `변경 기록`에서 확인할 수 있습니다.
